# 项目整理说明

本项目已精简为“评分专用流程”，保留评分主脚本和核心计算逻辑，删除了多余的演示、旧脚本与文档。

## 当前保留文件

- `run_scoring_with_dialog.py`：评分主脚本（带文件选择与进度提示）
- `single_sheet_processor_v2.py`：评分计算核心逻辑
- `README.md`：使用说明
- 数据文件与评分结果（视需要保留）

## 使用方式

1. 运行评分脚本：
```bash
python run_scoring_with_dialog.py
```

2. 选择一个或多个 CSV/XLSX 文件，生成带 `_评分合并` 后缀的结果文件。

## 评分汇总规则

- `SYNTAX_score`：按患者所有病变求和
- `Gensini_score`：按患者所有病变求和
- `CAD_RADS_grade`：按患者取最大值
- `SYNTAX_class` / `Gensini_class`：按总分重新分层

## 隐私建议

请勿将包含患者信息的数据文件提交到版本控制系统。
